# ratingProduct

The following are the four endpoints. 
1. For adding the products. when a shopkeeper or site's admin will be adding the product to the website.
2. For buying the product, Whenever a customer buys some product, his/her id will be stored in the customers' array of that product.
3. For rating that product, whenever a person tries to rate some product, our system will check if this person has brought this product or not(by checking in the customer array of that product), if that person hasn't bought that product, he will not be able to rate that product. Otherwise, he can give a rating to it from 1-5.
4. For showing our product, we will show the details of our product with the avg. rating plus a count of each rating.
